1.키워드란 무엇인가?
예약어, 사용목적 이외의 용도로 쓸 수 없다

2.2의 보수법이란?
2진수로 음수를 표현하기 위해 사용하는 반대되는 수

3.컴퓨터에서 음의 정수 표현방법에 대하여 설명하시오.
양의 정수의 이진표현에 2의 보수를 취한 결과

4.-5가 컴퓨터에서의 2진수 값을 나타내시오.
11111011

5.실수에서 오차가 생기는 원인은 무엇인가?
소수 부분을 나타내는 비트가 정확한 값을 표현할 수 없기 때문

6.실수의 표현법은(고정소수법, 부동소수법)을 설명하시오.
고정 = 소수점이 고정된 상태
부동 = 소수점의 이동이 가능한 상태

7.정밀도란 무엇인가요?
실수를 표현할 때 값의 정확도(실수는 정밀도를 낮추고 표현할 수 있는 값의 범위를 넓혔다)

8.상수란 무엇이며 변수와의 차이는?
변수에 값을 딱 한번만 할당할 수 있고 변경이 불가능한 것, 키워드 final선언이 붙어있는 변수

9.초기화란 무엇인가?
변수를 선언할 때 변수의 값을 지정해주는 것

10.리터럴이란 무엇인가요?
자료형을 기반으로 표현되는 상수

11.아래의 내용이 에러가 나는 이유와 수정방법은?
System.out.println(3147483647 + 3147483648);

12.자동 형변환과 명시적 형변환에 대하여 설명하시오.
자동 = 자료형의 크기가 큰 방향으로 형이 변환됨
명시적 = 자동 형변환 규칙에 부합하진 않지만 형변환이 필요한 상황이면 명시적으로 형을 변환한다

13.아래의 에러가 나는 이유를 설명하고 수정하시오.
    	int a = 3;
    	int b = 4;
    	
    	double result = a / b;
    	
    	System.out.println(result);

->int형끼리의 계산을 double형에 저장해도 int형으로 계산된 후 저장되기 때문에 0.75가 아닌 0.0이 출력된다.

14.이스퀘이프 시퀀스의 종료를 나열하고 기능을 설명하시오.
\b = 백스페이스, \t = 탭, \\  = \, \' =작은 따옴표, \" = 큰 따옴표, \n = 개행, \r = 캐리지리턴

15.아래의 출력결과를 확인하고, 130이 나오는 이유를 설명하시오.
System.out.println('A' + 'A');
A의 아스키코드 값은 65라서 65+65한 값인 130이 나오게된다.
