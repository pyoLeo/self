1.HelloWorld 를 출력하는 프로그램의 과정을 설명하시오.
.java가 javac.exe를 통해 컴파일 동명의.class가 생성되어 java.exe로 실행. 클래스 로더를 통해 .class를 JVM으로 가져와 byte code 검증 후 인터프리터가 byte코드를 binary코드로 변환해준다

2.아래의 명령어를 설명하시오(뭐하는건가).
-javac
자바 컴파일러 소스파일(.java)을 바이트코드인 클래스파일(.class)로 컴파일 해줌

-java
자바 프로그램을 실행시켜주는 명령어

3.컴파일이란 무엇인가요?
고급 언어로 쓰인 프로그램을 컴퓨터가 실행할 수 있는 기계어로 번역하여 프로그램을 고치는 과정

4.java 언어를 창시한 사람은?
제임스 고슬링

5.JDK란 무엇이며, 어디서 다운로드 받으며, OS별로 버전이 있는 까닭은?
Java Development Kit 오라클 홈페이지에서 무료로 다운이 가능하고 운영체제마다 다른 JVM이 제공되기 때문이다

6.JVM 무엇의 약자이며,이란 무엇인가?
Java Virtual Machine 스택기반의 자바가상머신 java와 os 사이에서 중개자 역할을 하며 java가 os에 구애받지 않고 재사용할 수 있도록 해준다
